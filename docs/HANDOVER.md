# 引継書

## 完成状況

Ver.1.1のモックモード主要導線は完成。Ver.1機能に加え、おすすめジャンルの追加・削除・編集・表示制御・ドラッグ並び替え・初期化・利用履歴、および記事URL優先・正規化・判定・コピー・検索補助が動作する。旧ローカルデータは初回読込時に互換移行する。

## 未対応

- OpenAI APIのサーバー接続とZod検証
- 正式RSS/ニュースAPI接続、実フィードでのリダイレクト追跡運用、関連記事の重複クラスタリング
- D1/R2による複数端末同期
- ユーザー写真のクロップ・拡大縮小・明るさ編集
- 縦長1080×1350と通知

## 既知の問題

- Ver.1はモックニュースと端末内保存。ブラウザーのサイトデータを消すと下書きも消える。
- Safariのクリップボード・ダウンロードは端末設定により確認を求める場合がある。
- ニュースの鮮度はモックデータ更新に依存する。
- モックニュースの多くは実在しないニュース内容のため、直接記事URLを持たず検索補助を表示する。
