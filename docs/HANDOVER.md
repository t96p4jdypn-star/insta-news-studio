# 引継書

## 完成状況

Ver.1.2の生成AIなし主要導線は完成。Ver.1.1機能に加え、タイトルによる記事検索、記事URL付き手動登録、画像内テキスト・4レイアウト・順番・複製・削除、文章ブロックの並び替え・ON/OFF・整形・短縮が動作する。旧ローカルデータは互換移行する。

## 未対応

- OpenAI APIのサーバー接続とZod検証
- 正式RSS/ニュースAPI接続、実フィードでのリダイレクト追跡運用、関連記事の重複クラスタリング
- D1/R2による複数端末同期
- ユーザー写真のクロップ・拡大縮小・明るさ編集
- 縦長1080×1350と通知

## 既知の問題

- Ver.1はモックニュースと端末内保存。ブラウザーのサイトデータを消すと下書きも消える。
- Safariのクリップボード・ダウンロードは端末設定により確認を求める場合がある。
- ニュースの鮮度はモックデータ更新に依存する。
- モックニュースの多くは実在しないニュース内容のため、直接記事URLの代わりにタイトル検索を表示する。実記事はユーザーがURLを貼って登録できる。
