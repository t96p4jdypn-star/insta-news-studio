# 仕様概要

正式仕様はユーザー提示「開発仕様書・Codex実装指示書 Ver.1.0」。本実装の中心は、ニュース選択→コメント→投稿素材生成→画像保存→本文コピー→Instagramで最終投稿、という初心者向け導線である。

## Ver.1受入条件

- PC/iPhoneレスポンシブ、PWA
- 興味登録、ニュース候補、投稿候補化
- コメント、スタイル、文体、1〜5枚生成
- 1080×1080 PNG、本文コピー、投稿ガイド
- 下書き、完成、投稿済み、カレンダー、検索
- API未設定のモックモード、ビルド・自動テスト

ニュース本文・写真は転載せず、見出し、短い要約、配信元、公開日時、リンクだけを扱う。
