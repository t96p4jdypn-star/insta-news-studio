# 変更履歴

## 1.3.0 - 2026-07-15

- Google News公開RSSからジャンル別の実ニュースをサーバー側で取得
- 公開環境でGoogle Newsへ接続できない場合のBing News RSS自動フォールバックと直接記事URL抽出を追加
- 検索・除外キーワード連携、最大10件への簡易重複排除を追加
- 実ニュース・取得中・モック予備の接続状態と手動更新ボタンを追加
- RSS成功結果を端末へ保存し、通信失敗時は保存済みニュースまたはモックへ自動フォールバック
- Google Newsの記事ラッパーURLを記事リンクとして正しく判定
- RSS解析、入力制限、除外、重複排除、画面フォールバックの自動テストを追加
- 生成AI・APIキーなしの設計を維持

## 1.2.0 - 2026-07-15

- 記事リンク未登録ニュースへ「タイトルで記事を探す」を追加
- 記事本文URL、タイトル、配信元、要約、ジャンルを入力する手動ニュース登録と端末保存を追加
- 画像ごとの見出し・本文、4種類のレイアウト、前後移動、複製、削除を追加
- Canvas PNGへ編集後の文字とレイアウトを反映
- 見出し、概要、意見、注目点、問いかけ、出典、ハッシュタグの文章構成エディターを追加
- 文章ブロックの並び替え、表示切替、直接編集、ルール式整形・短縮・初期化を追加
- すべて生成AI・外部APIなしで動作する設計を維持

## 1.1.0 - 2026-07-14

- おすすめジャンルの名称、アイコン、説明、検索・除外キーワード、表示順、表示可否、初期表示対象を管理可能に変更
- ジャンル追加・削除、PCドラッグ、スマートフォン長押しドラッグ、初期8ジャンルへの確認付きリセットを追加
- 利用回数と最終利用日を記録し、最近よく使うジャンルを確認後におすすめへ追加する導線を追加
- NewsItemへarticleUrl、sourceUrl、feedUrl、originalUrl、isArticleUrl、redirectCountを追加
- 計測パラメータ除去、記事ページ判定、RSSリダイレクト追跡境界を追加
- ニュースカードと編集画面を「この記事を読む」「配信元を見る」「リンクをコピー」「タイトルをコピー」へ改善
- 旧プロフィールを失わないローカルデータ移行と追加自動テストを実装

## 1.0.0 - 2026-07-14

- 初回設定、モックニュース、ニュース選択を実装
- コメント、スタイル、文体、8テーマを実装
- 1〜5枚の1080×1080 Canvas画像とPNG保存を実装
- 安全な固定ルール本文、コピー、出典表記を実装
- ストック、検索、月間カレンダー、投稿済み状態を実装
- Instagram初心者ガイド、PWA、データエクスポートを実装
- レスポンシブUI、自動テスト、CI、ドキュメントを追加
