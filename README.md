# Insta News Studio

ニュースから今日の投稿をつくる、Instagram初心者向けの投稿素材作成Webアプリです。気になるニュースを選び、自分の短いコメントを添えると、正方形画像（1〜5枚）とキャプションを生成します。Ver.1.2は生成AIや外部APIなしで、記事登録、画像編集、文章構成まで動きます。

## ユーザー向け使い方

1. 初回画面で利用目的と興味ジャンルを選ぶ
2. 「今日のおすすめ」からニュースを選ぶ
3. 自分のコメントを短く入力する
4. 投稿スタイル、文体、画像枚数、デザインを選ぶ
5. 「投稿素材を生成する」を押す
6. 画像をPNG保存し、本文をコピーする
7. 投稿ガイドを見ながらInstagramで最終投稿する
8. 投稿後にストック画面で「投稿済みにする」を押す

## Instagram初心者向け操作

Instagramで「＋」→「投稿」→保存した画像を選択→「次へ」→キャプション欄を長押し→「貼り付け」→内容確認→「シェア」の順です。最後のシェアは必ずユーザー自身が押します。投稿前に名前、数字、誤字、出典、画像の利用権限を確認してください。

## 現在の機能

- 興味ジャンル・利用目的・除外キーワード設定
- おすすめジャンルの追加・削除・名称・アイコン・説明・表示順・表示ON/OFF管理
- PCドラッグ、スマートフォン長押しドラッグ、利用回数、最近よく使うジャンルの提案
- モックニュースと交換可能なNewsProvider境界
- 記事URL・配信元URL・RSS URL・取得時URLの分離、URL正規化、記事ページ判定
- 「この記事を読む」「配信元を見る」「リンクをコピー」「タイトルをコピー」のニュース導線
- リンク未登録ニュースのタイトル検索と、記事本文URLを貼り付ける手動ニュース登録
- 5段階の興味度、あとで読む、興味なし
- コメント入力補助、投稿スタイル12種、文体7種
- デザインテンプレート8種、1〜5枚のカルーセル構成
- 各画像の見出し・本文・4レイアウト・順番・複製・削除の直接編集
- Canvasによる1080×1080 PNG生成・保存
- 固定ルールによる安全なキャプション、出典、ハッシュタグ生成
- 7つの文章ブロックを並べ替え、ON/OFF、加筆、整形、短縮できる文章構成エディター
- 投稿ストック、状態変更、月間カレンダー、全文検索
- PWAマニフェストとサービスワーカー
- JSONエクスポート、Instagram初心者ガイド
- localStorage保存と将来の同期先に交換できるデータ境界

## 開発環境

- vinext / Next.js 16 / React 19 / TypeScript / Tailwind CSS 4
- Cloudflare Workers互換ビルド
- Node.js 22以上、pnpm 11

```bash
pnpm install
pnpm dev
```

## 環境変数

`.env.example`を参照してください。`OPENAI_API_KEY`などの秘密情報はサーバー側のみで使用し、`NEXT_PUBLIC_`を付けません。現在は未設定でも全主要機能が動作します。

## ニュース取得方法

`src/services/news/`にプロバイダー境界があります。Ver.1.2は`MockNewsProvider`と端末内の手動登録を使用し、正式利用可能なRSSは`RssNewsProvider`へ差し替えられます。`url.ts`がリダイレクト追跡、計測パラメータ除去、記事ページ判定、タイトル検索リンクを担当します。本文やニュース画像は保持せず、見出し、短い要約、配信元、日時、リンクだけを扱います。

## 著作権・画像利用上の注意

ニュース本文の長文転載、ニュース写真の保存・加工・再投稿をしません。投稿画像は文字、図形、ユーザーが権利を持つ写真などで構成します。ユーザー画像は、自分で撮影したか使用許可を得たものだけを利用してください。生成キャプションはニュースの概要と個人の意見を分け、配信元を明示します。
